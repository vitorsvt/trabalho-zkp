# Zero Knowledge Proofs

Para executar, utilize o seguinte comando no diretório raiz do projeto (que contém o `README.md` e o diretório `zkp/`):

```sh
python -m zkp
```

Os experimentos 4 e 5 foram desativados por padrão devido ao tempo necessário para a execução. Para ativar, utilize os seguintes flags na linha de comando:

- Experimento 4: `--run_perf`, exemplo:
    ```sh
    python -m zkp --run_perf
    ```

- Experimento 5: `--run_perf`, exemplo:
    ```sh
    python -m zkp --run_robust
    ```

Observação: ambos os flags podem estar ativos de uma vez.
